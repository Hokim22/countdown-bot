# システム概要スライド - 原稿

## スライド: システム概要
### 【原稿】

「それでは、カウントダウンbotシステムの概要についてご説明します。

このシステムは、試験や発表などの重要なイベントまでの日数をカウントダウンし、AIが生成した応援メッセージをSlackに投稿するシステムです。

システムの主要機能は5つあります。

**まず、カウントダウン機能です。**
設定された目標日までの残り日数を自動で計算し、毎日定時に実行されます。現在はテスト用に1時間間隔で動作していますが、本番運用では毎日朝7時30分に実行する予定です。

**次に、キャラクター応援機能です。**
8種類のキャラクターを事前に設定しており、毎回ランダムに選択されます。天真爛漫な友人、毒舌系フリーター、優しいお姉さん、推しのアイドルなど、個性豊かなキャラクターが日替わりで応援してくれるため、ユーザーが飽きることなく継続的に利用できます。

**3つ目は、AI メッセージ生成機能です。**
Amazon Bedrockを活用して、選択されたキャラクターの個性に合わせたメッセージを生成します。単なる励ましの言葉だけでなく、プレゼンテーションの準備方法や時間管理など、具体的で実用的なアドバイスも含まれています。毎回異なる表現で生成されるため、同じメッセージが繰り返されることはありません。

**4つ目は、通知機能です。**
生成されたメッセージは、Slack Webhook APIを通じて指定されたチャンネルに自動投稿されます。スマートフォンでもリアルタイムに通知を受け取ることができ、どこにいても応援メッセージを確認できます。

**最後に、データ管理機能です。**
試験名、目標日、通知先などの情報をDynamoDBに保存し、複数の試験や発表を同時に管理することができます。設定の変更も簡単で、新しい試験の追加や日程変更にも柔軟に対応できます。

これらの機能により、個人の学習や仕事のモチベーション維持を、楽しく効果的にサポートするシステムとなっています。

次に、具体的な技術構成について詳しく説明していきます。」
