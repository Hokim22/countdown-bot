# PowerPoint用スライド構成

## スライド1: タイトル
```
カウントダウンbotシステム
〜試験・発表応援AI bot〜

システム構成図
```

## スライド2: システム概要
```
【システム概要】

✅ 毎日朝7:30に自動実行
✅ 8種類のキャラクターがランダム応援
✅ AI生成による個性的なメッセージ
✅ 実用的なアドバイス付き
✅ 完全サーバーレス構成
✅ 月額5-9ドルの低コスト運用
```

## スライド3: アーキテクチャ図
```
EventBridge          Lambda              DynamoDB
┌─────────┐         ┌─────────┐         ┌─────────┐
│⏰ 毎日  │────────▶│countdown│────────▶│試験     │
│  7:30   │         │  -bot   │         │データ   │
│cron実行 │         └────┬────┘         └─────────┘
└─────────┘              │
                         ▼
                   ┌─────────┐
                   │Bedrock  │
                   │Claude 3 │
                   │Haiku    │
                   └────┬────┘
                        ▼
                   ┌─────────┐
                   │ Slack   │
                   │Webhook  │
                   └─────────┘
```

## スライド4: データフロー
```
【処理フロー】

1️⃣ EventBridge → 毎日7:30に起動

2️⃣ Lambda → DynamoDBから試験データ取得

3️⃣ 日数計算 & キャラクター選択（8種類）

4️⃣ Bedrock AI → キャラ別メッセージ生成

5️⃣ Slack → Webhook経由でメッセージ投稿
```

## スライド5: キャラクターシステム
```
【8種類のキャラクター】

🌟 天真爛漫な友人     😏 毒舌系フリーター
🌸 優しいお姉さん     💖 推しのアイドル
⭐ 未来の自分        🔥 競い合うライバル
🙏 守護霊           🌿 温かい家族

毎日ランダム選択で飽きない応援！
```

## スライド6: 技術スタック
```
【使用技術】

スケジューラ    │ Amazon EventBridge
コンピュート    │ AWS Lambda (Node.js)
データベース    │ Amazon DynamoDB
AI生成         │ Amazon Bedrock (Claude 3 Haiku)
通知           │ Slack Webhook API
権限管理       │ AWS IAM
監視           │ Amazon CloudWatch
```
